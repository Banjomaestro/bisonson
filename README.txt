

Pour lancer notre projet vous aurez besoin d'adapter notre makefile afin qu'il 

puisse trouver les différentes librairies nécessaires dans la bonne version.

Voici la liste des librairies utilisées : 

- sdl 2.0.20
- sdl2_image2.0.5
- sdl2_ttf 2.0.18_1
- freetype 2.12.0
- sfml 2.5.1_1
- framework GLUT 
- framework OpenGL
